## 🚨 Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| **"Cannot connect to database"** | Check MongoDB Atlas cluster is active |
| **"DNS Error" on startup** | Automatic DNS retry should fix it |
| **"Invalid token"** | Clear localStorage, login again |
| **"Port 5000 in use"** | Kill process or use different port |
| **"Module not found"** | Run `npm install` in both directories |
| **"Blank page on load"** | Hard refresh (Ctrl+Shift+R) |
